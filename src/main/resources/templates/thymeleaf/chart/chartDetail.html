미수거래 설명

미수거래는 매수 시 구매 금액의 일부(예: 30%)만 현금으로 납부하고, 나머지 금액은 결제일(T+2일)까지 납부하는 거래 방식입니다.
미수금을 결제일까지 납부하지 않으면 외상구매 정지계좌로 지정될 수 있습니다.

지정가 설명

지정가는 사용자가 원하는 가격에 매수하거나 매도할 수 있도록 설정하는 주문 유형입니다. 설정한 가격에 도달하면 자동으로 체결됩니다.

구매 가능 금액 설명

구매 가능 금액은 계좌 잔고와 미수거래 가능 금액을 포함하여 계산된 최대 매수 가능 금액입니다.
거래 수수료와 세금을 포함하여 계산된 금액으로 표시됩니다.

총 주문 금액 설명

총 주문 금액은 입력한 매수 금액과 수량을 곱한 값입니다. 해당 금액에는 거래 수수료 및 세금이 포함될 수 있습니다.

수량 설명

수량 입력란에 원하는 값을 입력하거나, + 또는 - 버튼으로 조정할 수 있습니다.

시장가 설명

시장가는 현재 시장에서 거래 가능한 가장 유리한 가격으로 주문이 체결됩니다. 구매 시 +3%, 판매 시 -3% 범위로 가격이 계산될 수 있습니다.

총 수익 설명

총 수익은 보유한 주식의 현재 평가 금액과 총 매입 금액 간의 차이를 나타냅니다.
수익률(%)은 이를 퍼센트로 환산한 값이며, '수수료·세금'을 체크하면 실질적인 수익을 확인할 수 있습니다.

총 금액 설명

총 금액은 보유한 주식 수량과 현재 주가를 곱한 값입니다.
이는 주식의 현재 가치를 빠르게 확인할 수 있는 지표입니다.

수량 설명

수량은 현재 보유 중인 주식의 개수를 나타냅니다. 1주는 1개의 주식을 의미합니다.

1주 평균 금액 설명

1주 평균 금액은 보유한 주식의 평균 매입 단가를 나타냅니다. 여러 번 매수한 경우, 총 매입 금액을 매수한 주식 수로 나눈 값입니다.

시간 간격 설명

시간 간격은 차트에 표시되는 데이터의 시간 단위를 설정합니다. 예를 들어, '1분'을 선택하면 1분 단위로 데이터가 그룹화되어 표시됩니다. 이를 통해 짧은 시간 간격의 변화를 빠르게 파악할 수 있습니다.

시간 프레임 설명

시간 프레임은 차트에서 데이터를 그룹화하는 단위를 설정합니다. 예를 들어 '일'을 선택하면 하루 단위로 데이터를 확인할 수 있으며, '주', '월', '년'을 선택하면 각 단위별 요약된 데이터를 볼 수 있습니다.

보조지표 설명

보조지표는 주식 차트 분석에 사용되는 도구입니다. 이동 평균선, RSI, MACD 등과 같은 다양한 보조지표를 추가하여 주식 가격의 흐름과 거래량을 분석하고 향후 움직임을 예측할 수 있습니다.

그리기 설명

그리기 기능은 차트에 선, 화살표, 텍스트 등을 추가하여 개인적인 분석과 메모를 기록하는 데 유용합니다. 중요한 가격 지점이나 추세를 시각적으로 표시할 수 있습니다.

종목비교 설명

종목비교는 선택한 주식과 다른 종목의 주요 지표를 비교하여 투자 결정을 도와줍니다.

차트모양 설명

차트모양은 차트의 스타일을 변경하는 기능입니다. 예를 들어, 캔들 차트, 선형 차트 등 다양한 모양으로 차트를 시각화할 수 있습니다.

종목 정보 설명

종목 정보는 선택한 주식의 기본적인 정보를 제공합니다. 회사 개요, 업종, 주요 지표 등 투자에 필요한 데이터를 확인할 수 있습니다.

뉴스 / 공시 설명

뉴스 / 공시는 해당 종목과 관련된 최신 뉴스와 공시 정보를 확인할 수 있는 영역입니다. 이 정보를 통해 기업의 주요 활동과 시장 동향을 파악할 수 있습니다.

커뮤니티 설명

커뮤니티는 투자자 간의 소통 공간으로, 질문, 꿀팁 공유, 수익 인증 등 다양한 정보를 교류할 수 있는 곳입니다.

주식 이름 설명

이 영역은 현재 보고 있는 주식이나 기업의 이름을 나타냅니다. 해당 주식의 정보 및 관련 데이터를 보여주고 있습니다.

주식 코드 설명

주식 코드는 기업의 주식을 식별하기 위해 사용되는 고유한 번호입니다. 예를 들어, 삼성전자의 경우 "005930"이 주식 코드로 사용됩니다.

실시간 주식 가격 설명

실시간 주식 가격은 현재 시장에서 거래되고 있는 해당 주식의 가격을 나타냅니다. 이 가격은 매수자와 매도자 간의 거래에 따라 실시간으로 변동됩니다.

차트 제목 설명

차트는 선택한 주식의 가격 변화, 거래량, 보조 지표 등을 시각적으로 확인할 수 있는 도구입니다. 이를 통해 시장의 추세를 분석하고 투자 결정을 내릴 수 있습니다.

체결가 설명

체결가는 주식 거래가 실제로 이루어진 가격을 의미합니다. 매수자와 매도자의 주문이 일치하여 거래가 성사된 시점의 가격입니다.

체결량 설명

체결량은 특정 시간 동안 주식이 거래된 총 수량을 나타냅니다.

등락률 설명

등락률은 주식의 현재 가격이 전일 대비 얼마나 상승하거나 하락했는지를 백분율로 나타냅니다.

거래량 (주) 설명

거래량은 특정 시간 동안 주식이 매매된 총 수량을 나타냅니다. 거래량은 시장의 활동성과 주식의 유동성을 판단하는 데 유용합니다.

시간 설명

시간은 주식 체결이 발생한 정확한 시점을 나타냅니다. 이를 통해 거래가 이루어진 시간대를 확인할 수 있습니다.

내 주식 평균 설명

내 주식 평균은 보유한 주식의 평균 매수 단가를 나타냅니다. 이를 통해 현재 주가와 비교하여 손익을 계산할 수 있습니다.

구매 후 예상 설명

구매 후 예상은 현재 매수를 진행한 후 예상되는 손익과 포트폴리오 상태를 나타냅니다. 이는 구매 후 투자 상황을 가늠하는 데 도움을 줍니다.

대기 설명

'대기'는 아직 체결되지 않은 주문들을 나타냅니다. 지정가 주문이나 조건부 주문이 해당 상태에 포함될 수 있으며, 조건이 충족될 때까지 체결이 보류됩니다.

완료 설명

'완료'는 체결이 완료된 주문을 확인하는 탭입니다. 매수 또는 매도가 완료된 내역을 확인할 수 있습니다.

조건주문 설명

'조건주문'은 특정 조건(가격, 시간, 거래량 등)을 설정하여 조건이 충족되었을 때 자동으로 주문이 실행되도록 설정하는 기능입니다.

가격 입력 설명

가격 입력은 매수나 매도 시 사용자가 원하는 가격을 설정하는 영역입니다.
지정가: 사용자가 입력한 가격에 도달했을 때 거래가 체결됩니다.
시장가: 현재 시장에서 가능한 가장 유리한 가격으로 즉시 거래가 체결됩니다.
버튼을 클릭하거나 직접 입력하여 금액을 조정할 수 있습니다.

지정가 설명

지정가는 사용자가 원하는 가격에 주식을 매도할 수 있도록 설정하는 주문 방식입니다.
설정한 가격에 도달할 때만 매도 거래가 체결됩니다.

시장가 설명

시장가는 현재 시장에서 형성된 가격으로 즉시 매도 주문을 실행하는 방식입니다.
체결 속도가 빠르지만, 정확한 가격을 보장하지는 않습니다.

주문 유형 설명

- 일반 주문: 현재 시장 가격에 따라 즉시 체결됩니다.
- 조건 주문: 사용자가 설정한 조건을 충족할 때 자동으로 체결됩니다.

예상 수익률 설명

예상 수익률은 현재 가격과 지정한 가격의 차이를 백분율로 계산한 값입니다.
실제 체결 시 수수료 등이 반영되어 변동될 수 있습니다.

예상 손익 설명

예상 손익은 현재 가격과 구매 가격, 판매 가격의 차이를 바탕으로 계산한 금액입니다.
실제 체결 시 수수료 등이 반영되어 변동될 수 있습니다.

총 금액 설명

총 금액은 입력한 수량과 지정가(또는 시장가)의 곱에 수수료가 포함된 금액입니다.
실제 체결 금액은 변동될 수 있습니다.

비율 설명

버튼을 클릭하면 계좌에 있는 자산 중 선택한 비율(10%, 25%, 50%, 최대)에 해당하는 금액을 자동으로 계산하여 체결 금액으로 설정합니다. 체결 금액을 빠르게 설정하고 싶을 때 유용합니다.
